# 财富创造

> 赚钱不是一件想做就能做的事情，而是一门需要学习的技能。

## 核心框架：如何致富（不靠运气）

这是纳瓦尔最著名的推特风暴，是他所有理念和原则的精髓。如果你能吸收这些理念，以此为指引，奋斗10年，就一定能够得偿所愿。

### 财富的本质

- **追求财富，而不是金钱或地位**
  - 财富是指在你睡觉时仍能为你赚钱的资产
  - 金钱是我们转换时间和财富的方式，是社会的信用符号
  - 地位是你在社会等级体系中所处的位置

- **财富的定义**：在睡觉时也能带来收入的企业和资产
  - 可以是进行生产的工厂和机器人
  - 可以是不分昼夜为客户服务的计算机程序
  - 可以是银行里被投资于其他资产或业务的钱

- **拥有股权是关键**
  - 依靠出租时间是不可能致富的
  - 必须拥有股权（企业的部分所有权），才能实现财务自由
  - 真正的财富是通过创建公司或者通过投资创造出来的

### 专长与独特性

- **专长的定义**：无法通过培训获得的知识
  - 如果社会可以培训你，那么社会也可以培训他人来取代你
  - 专长属于知识领域的边缘地带
  - 专长往往具有高度的技术性或创造性，不能被外包或自动化

- **找到你的专长**
  - 回想孩提时代或青少年时期，哪些事情是你几乎不费吹灰之力就可以完成的
  - 专长是一个人个性和身份的有机组成部分
  - 在"成为自己"这件事情上，没有人能比得过你

- **独辟蹊径，避开竞争**
  - 竞争就是模仿，是因为你跟别人在做一样的事情
  - 如果你打造和推介的内容是自我的衍生，那就没有人可以与你竞争

### 杠杆效应

纳瓦尔将杠杆分为三种形式：

#### 1. 劳动力杠杆

- 让别人给你打工
- 最古老、争夺最激烈的一种杠杆
- 管理他人是非常复杂、极具挑战的工作

#### 2. 资本杠杆

- 用钱来扩大决策的影响力
- 管理资本要比管理人更容易
- 20世纪杠杆的主要形式

#### 3. 代码和媒体（新富阶层背后的杠杆）

- 复制边际成本为零的产品
- **不需要许可就能使用的杠杆**
- 包括书籍、媒体、电影、代码
- 如果不会写代码，那就出书、写博客、做视频、录播客

> 现代人的二分法是"利用了杠杆的人"和"没有利用杠杆的人"。

### 判断力

- **努力的作用被大大高估了**
  - 在现代经济中，工作的努力程度并没有那么重要
  - 判断力才是被低估的关键

- **智慧与判断力**
  - 对智慧的定义是"知道个人行为的长期后果"
  - 用于解决外部问题的智慧其实就是判断力

- **在杠杆时代**
  - 一个正确的决策可以帮你赢得一切
  - 方向比速度更重要
  - 在每个岔路口选对方向，其重要程度要远远超过前进的努力程度

### 分清主次，聚焦重点

- **给自己设定极高的时薪**
  - 即使在我年轻的时候，我也认为自己的时间比市场的定价更值钱
  - 从时间成本的角度做决策
  - 如果用花钱的方式节省的时间价值更高，那就花钱

- **人生早期的三个重大决定**
  1. 在哪里生活
  2. 和谁在一起
  3. 从事什么职业

- **零和游戏 vs 正和游戏**
  - 金钱游戏是正和游戏，我们一起把蛋糕越做越大
  - 地位游戏是零和游戏，你要击败他人
  - 避免在生活中玩地位游戏

### 找到如玩耍般的工作

- **退休的定义**：不再为了想象中的明天而牺牲今天
- **达到退休状态的三种方法**：
  1. 存钱，让被动收入满足开销
  2. 把开销降为零——出家修行
  3. 做自己热爱的事情，完全乐在其中

- **创业就是玩耍**
  - 我从创业中获得的报酬是副产品
  - 以前创业是为了实现某个目标，现在是为了把企业做到极致，使之成为精美的艺术品

- **对钱的贪欲**
  - 钱本身是无罪的，真正有害的是对钱的贪欲
  - 摆脱金钱贪念最好的办法：赚钱后不要升级自己的生活方式

### 如何获得运气

纳瓦尔将运气分为四种：

1. **不期而遇的运气**：完全源于你控制范围之外的事情
2. **源于坚持不懈的运气**：靠个人主动创造机会获得
3. **善于发现好运**：幸运会眷顾有准备的大脑
4. **让运气找到你**：打造独特的个性、独特的品牌、独特的心态

> 人品和声誉是可以自己建立的。一旦有了人品和声誉，你就会有很多机会。别人认为你是运气好，但你知道那不是运气，而是人品。

### 保持耐心

- **成功需要时间**
  - 即使万事俱备，需要投入的时间也具有不确定性
  - 如果一直在掐算时间，在成功真正到来之前，你的耐心就会被消磨殆尽

- **大自然的法则**
  - 春播秋收，极少有事情是立竿见影的
  - 不要去计算自己投入的时间和精力，因为一旦开始计算，你就会失去耐心

- **"但行好事，莫问前程"**
  - 这是人生中最难做到的一件事，但也是让人收获最大的一件事
  - 人生真正的履历，其实就是一生所承受痛苦的集合

## 📌 访谈补充：成功与耐心

> 以下内容基于 Naval Ravikant 与 Shane Parrish 的深度访谈（The Knowledge Project，2020）提炼。

### 概念：十年投入是成功的必要条件

**来源**：The Knowledge Project - Naval Ravikant Interview (2020)

**核心观点**：成功需要至少十年的投入。你必须真正热爱你所做的事情，愿意在没有回报之前就投入时间。最优秀的创始人对领域有深刻理解，具备逆向思维，充满热情，并且有执行力。

**纳瓦尔原话**：

> “去找一件你能投入十年的事情，因为至少需要十年才能取得好的结果。你必须享受这个过程，因为结果无法保证。你必须非常非常擅长它，这意味着你可能非常热爱它，以至于愿意在没有任何回报之前就投入时间。”

**延伸思考 / Q/A**：

- **问**：如何判断自己是否真的热爱某件事？
- **答**：如果你愿意在没有任何回报之前就投入时间，那就是真正的热爱。如果你只关心结果，那可能不是。

**关联主题**：`#成功` `#耐心` `#复利`

### 概念：优秀的人会带来优秀的成果，要有耐心

**来源**：The Knowledge Project - Naval Ravikant Interview (2020)

**核心观点**：纳瓦尔回顾20年前职业生涯初期遇到的优秀年轻人，他们无一例外都取得了巨大成功。成功永远不会在你或他们想要的时间内发生，但它确实会发生。

**纳瓦尔原话**：

> “20年前，我职业生涯初期遇到的每个人，我当时都会说：‘哇，这个人或那个女孩真是太有能力了，他们非常聪明、非常敬业。’结果，几乎所有人都取得了巨大的成功。你只需要给他们足够的时间。成功永远不会在你或他们想要的时间内发生，但它确实会发生。”

**延伸思考 / Q/A**：

- **问**：如何识别有潜力的人？
- **答**：寻找那些对领域有深刻理解、充满热情、愿意长期投入、有执行力的人。

## 核心概念卡片

### 概念：把自己产品化

**来源**：《纳瓦尔宝典》第一章

**核心摘要**：这句话有两个重点，一个是"自己"，一个是"产品化"。"自己"具有独特性，"产品化"是发挥杠杆效应；"自己"具有责任感，"产品化"需要专长。

**Q/A 对**：

- **问**：如何实现"把自己产品化"？
- **答**：先问自己"这是我真正想要的东西吗？"得到肯定后，再问"我实现产品化了吗？我实现规模化了吗？我选择规模化的方式是劳动力、资本，还是代码或媒体？"

### 概念：特定知识

**来源**：《纳瓦尔宝典》第一章

**核心摘要**：专长指的是无法通过培训获得的知识。如果社会可以培训你，那么社会也可以培训他人来取代你。

**纳瓦尔原话**：

> 专长无法被教授，但可以被学习。

**Q/A 对**：

- **问**：如何找到自己的特定知识？
- **答**：回想一下在孩提时代或青少年时期，有哪些事情是你几乎不费吹灰之力就可以完成的。专长是一个人个性和身份的有机组成部分。

### 概念：三种杠杆

**来源**：《纳瓦尔宝典》第一章

**核心摘要**：

- 劳动力杠杆：让别人给你打工
- 资本杠杆：用钱来扩大决策的影响力
- 代码和媒体：复制边际成本为零的产品，不需要许可就能使用

**纳瓦尔原话**：

> 现代人的二分法是"利用了杠杆的人"和"没有利用杠杆的人"。

### 概念：退休状态

**来源**：《纳瓦尔宝典》第一章

**核心摘要**：退休就是不再为了想象中的明天而牺牲今天。当你能活在当下，内心充盈地度过每一天时，你就达到了退休的状态。

**达到退休的三种方法**：

1. 存钱（被动收入满足开销）
2. 出家修行（把开销降为零）
3. 做自己热爱的事情

### 概念：四种运气

**来源**：《纳瓦尔宝典》第一章

**核心摘要**：

1. 不期而遇的运气
2. 源于坚持不懈的运气
3. 善于发现好运
4. 让运气找到你（打造独特的个性、品牌和心态）

**纳瓦尔原话**：

> 在一个长线游戏中，似乎每个人都在让彼此变得富有。而在一个短线游戏中，似乎每个人都在让自己变得富有。
